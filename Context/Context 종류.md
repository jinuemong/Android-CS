

### Context
- 애플리케이션 환경에 대한 전역 정보를 포함
- 여러 컴포넌트들의 상위 클래스
- 애플리케이션의 현재 상태를 나타냄
- 액티비티와 애플리케이션의 정보를 얻기 위해 사용할 수 있음
- 리소스, 데이터베이스, shared preference, 시스템 서비스 등에 접근하기 위해 사용할 수 있음

### Application Context
- 싱글턴 인스턴스로 애플리케이션 라이프사이클에 묶여있음

### Activity Context
- 액티비티 라이프사이클과 연결되어 있음
- 액티비티와 함께 소멸해야 하는 경우에 사용
